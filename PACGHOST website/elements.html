About PAC GHOST

PAC GHOST is currently in alpha 1.0.3

Our Idea

Our inpiration came from our teacher Mrs.Listman. We knew we wanted to create a game that could be played with more than one person. We started to think of multiplyer games that we could code in 12 hours. Thats when we just leaned how to implement keyboard listeners into JAVA.

Our Favorite Quote

Before software can be reusable it first has to be usable. (Ralph Johnson)

Our Driver Code


import javax.swing.JFrame;
public class DriverPAC
{
   public static void main(String[] args)
   { 
      JFrame frame = new JFrame("PACGHOST");
      frame.setSize(1000, 1000);
      frame.setLocation(0, 0);
      frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
      PACPanel p = new PACPanel();
      frame.setContentPane(p);
      p.requestFocus();
      frame.setVisible(true);
   }

}
